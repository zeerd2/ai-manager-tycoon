## 验收背景

R5-3 将以下 CSS 从 `App.css` 拆分到独立文件：

| 文件 | 行数 | 包含内容 |
|------|------|----------|
| `AgentCard.css` | 260 | 员工卡、锁定状态、进度条、技能树按钮 |
| `ProjectCard.css` | 231 | 项目卡、进度条、策略卡 |

## 六、发现视觉问题时的回补修复范围

如果上述任何检查项发现问题，修复范围限定为：

| 文件 | 说明 |
|------|------|
| `src/components/AgentCard.css` | 员工卡相关样式 |
| `src/components/ProjectCard.css` | 项目卡/策略卡相关样式 |

**不允许修改：**
- `src/App.css`（除非是遗漏的样式）
- `src/App.tsx`
- `src/components/*.tsx`（除非是 class 名不匹配）

**修复后必须重新跑：**
```bash
npm run lint
npm test
npm run build
```
